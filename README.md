# n8n-Automate-GitLab-tag-releases-to-Jira-and-Slack-for-Dev-and-QA
Automated GitLab Release Management workflow built with n8n, GitLab, Jira, and Slack. Detects Git tags, validates semantic versions, generates release notes from commits, creates Jira QA tasks, and sends release notifications to Dev and QA teams.
